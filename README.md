# DDD (Domain driven design)

Design dirigido à domínio.

## Domínio

- Domain Experts
  - Conversas com os especialistas do domínio
- Linguagem ubíqua
  - Linguagem comum entre os especialistas e os desenvolvedores

- Agregados
- Value Objects
- Eventos de Domínio
- Subdomínios (bounded contexts)
- Entidades
- Casos de uso
